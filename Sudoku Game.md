Sudoku Game

This will be a mobile first web application that allows users to play Sudoku. 
The game will be played on a 9x9 grid, with the objective of filling in the grid with numbers 1-9 such that each row, column, and 3x3 subgrid contains all the numbers from 1 to 9.

The game will have the following features:
- A 9x9 grid that is split into 9 3x3 subgrids.
- The grid will be filled with numbers 1-9, with some numbers already filled in.
- The user will be able to fill in the remaining numbers in the grid.
- The user will be able to check if their solution is correct.
- The user will be able to reset the game.

- The user will be able to play the game online.
- The user will be able to play the game with a timer.

- The user wil lbe able to see how quickly they solve a puzzle

- The user can select the difficulty of the puzzle

The interface should be modern and easy to use. The game should be responsive and work on all devices.
